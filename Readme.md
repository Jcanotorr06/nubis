# Nubis

Nubis is a Dex for the Polygon Network in development

## Installation

Use the package manager [yarn](https://yarnpkg.com/) to install nubis.

```bash
yarn 
```

## Run Dev Server

```bash
yarn start
```

## Build for Production

```bash
yarn build
```
